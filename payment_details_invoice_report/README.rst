Payment Details In Invoice Report
=================================
We can see the current payment state of the invoice in the report. We can add
all the payment details of a particular invoice while print the invoice. Here
we have the details such as Payment reference, Date of payment, Payment method
and Amount.

Configuration
=============
No additional configurations needed.
