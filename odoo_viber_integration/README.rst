Viber Integration
=================
This module helps to chat users with viber chatter app.

Features
========
* Create Reusable Message Templates for Consistent Content Distribution.
* Set Up User Groups for Simultaneous Message Sending.
* Schedule Messages.

Configuration
=============
* Download Viber from https://www.viber.com/en/download/
* Create a account in Viber.
