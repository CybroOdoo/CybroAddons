Overview

Allows waiter selection from pos interface and provides report of the performance of the waiter by analysing the sales carried by each waiter.

POS waiter performance analysis

  • The waiter selection can be enable/disable from point of sale settings.
  • Selected waiter will highlighted in pos ticket and order.
  • Option to distinguish normal employees and waiters.
  • Date wise report which shows the amount of sales carried by each waiters.

Screenshots

After installation, go to Point of sale > Enable Waiter Selection.

Move to employee form select the employee which we want to make as a waiter for that enable the field "Is a waiter"

Open the POS session, we can see there will be button named "Add Waiter" by click on the button we can see the option for selecting the waiters.

From this pop up window we can select our required waiters.

After the order confirmation, while printing the ticket the corresponding waiter will highlighted there.

When we moves to backend pos orders we can see waiter will highlighted there.

For analysing the performance of the waiter, the report wizard can be find under the menu Reporting->Waiter Performance Report

In the wizard we have to specify the period of the report by selecting start date and end date
