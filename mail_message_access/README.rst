Mail Message Access
===================
Implement the restriction on chatter messages for the users

Configuration
=============
In Odoo's chatter, you can restrict chatter messages for users by implementing access rights.
By setting specific access permissions, you can control who can send messages and what actions
they can perform within the chatter.
