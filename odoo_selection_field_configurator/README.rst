Odoo Selection Field Configurator
=================================
This module allows the user to add options to the selection fields in odoo.
This application is designed to have basic selection fields without any
additional functionalities. In order to add selections to core fields, it is
necessary to implement them through Python code, preferably by creating a
custom addon that provides the desired functionalities.

Installation
============
- www.odoo.com/documentation/15.0/setup/install.html
- Install our custom addon
