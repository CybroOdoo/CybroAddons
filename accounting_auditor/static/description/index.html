Accounting Auditor

Addition of a new group called Auditor.

Users in this group will be given read only permission.

Author : Cybrosys Techno Solutions , www.cybrosys.com

Features:

  • →  Only Read Permission
  • →   No Write Permission
  • →   No Create Permission

Overview

As Auditing is an important part of accounting we can create a new user who is having access right to the accounting module with read only permission. This provides the user a provision to read all the accounting related data but cannot edit or create any records.

How to create an auditor user

  • →From the Users form view (Settings -> User) , set the user as Auditor by selecting the Auditor in the Accounting & Finance
  • → Then remove the user from all the other groups

    • Audit User

  • →User is logged in as audit user
