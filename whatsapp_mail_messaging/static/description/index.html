Whatsapp Mail Messaging

Send Whatsapp messages and Mails from any view of Odoo

Key Highlights
  • Added Whatsapp and Mail icons in Systray for easy access from any views.
  • Can send mails to any valid mail addresses along with attachments.
  • Can send Whatsapp messages to any valid mobile numbers. along with country code.
  • After sending message from odoo then automatically redirected to Whatsapp web with our message.
  • Our module also supporting the emojis along with the whatsapp messages.

Overview

In this module we have added Whatsapp and Mail icon in the systray bar which the user can send messages through whatsapp and mail from any view using these icons.The mails will be sent from the mail which we configured as the outgoing mail server and in case of whatsapp message, it will redirected to the whatsapp web.

Features

  • Added Whatsapp and Mail icons in Systray for easy access from any views.
  • Can send mails to any valid mail addresses along with attachments.
  • Can send Whatsapp messages to any valid mobile numbers. along with country code.

  • Our module also supporting the emojis along with the whatsapp messages.

Screenshots

We will have a Mail icon in the systray bar after the installation of our module.

By Clicking on the Mail icon we will have the message composing wizard, from there we can select the recipient and can add the subject, content and attachments.

After click on sent the recipient will receive a mail like we composed from odoo. This mail will be sent from the outgoing mail server which we configured in the database.

We will have a Whatsapp icon in the case of sending Whatsapp messages to a number.

In case of selecting a contact directly for sending message the we have to fill the field mobile in the contact form.

After clicking on the whatsapp icon, we will have the message composing wizard from there we can select the contact, enter the mobile number along with the country code and content of the message along with emojis.

We can choose any of the emojis to our message listed below.

After click on send it will redirected to the whatsapp web with our message.

The message will available in the chat of the corresponding number in our contact.
