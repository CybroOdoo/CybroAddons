Email Verification For Partner And Employee
===========================================
With this plugin, you can validate the E-mail ID entered in Partner/Employee
form. This app will help you to maintain a consistent and reliable address
details for your Partner/Employee.

Configuration
=============
You must install following packages to work the App properly:
* pip install validate_email
* pip install py3DNS
