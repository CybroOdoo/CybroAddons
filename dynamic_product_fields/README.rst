Product Custom Fields
=====================
This module enables to generate custom fields in Product form.

Features
========
* Custom fields in Product form.
* Filter for dynamically generated fields in settings.

Configuration
=============
    - After installation, go to Settings - Users & Companies.
    - Select any user and enable the checkbox Create Custom Fields.
